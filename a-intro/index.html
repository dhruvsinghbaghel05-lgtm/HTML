INTRODUCTION TO HTML

  • HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages.

  • HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.

  • The component used to design the structure of websites are called HTML tags

  • HTML tags are keywords (tag names) surrounded by angle brackets: . An HTML tag can also come in pairs like: content . The first tag in a pair is the start tag, the second tag is the end tag (they are also called opening and closing tags).

  • These Tags are containers for some content.

  • HTML Attributes are used to provide additional information about HTML elements.

  • Quick Points To Be Remembered

    • HTML is not a programming language, it is a markup language.

    • HTML uses "tags" to create elements.

    • HTML documents are files that end with .html or .htm.

    • HTML tags normally come in pairs like: content .

    • The first tag in a pair is the start tag, the second tag is the end tag (they are also called opening and closing tags).

    • Some tags have no content in between, eg - br

    • We can use inspect element/view page source to edit html

  • Advantages of HTML

    • Easy to learn
    • Platform independent
    • Supported by all browsers
    • Works with CSS & JavaScript
